# Rules Module

Game rules engine and turn logic.

## Purpose

- Enforce game rules
- Manage turn sequence
- Detect win conditions
- Validate game state transitions

## Key Functions

- `isValidTurn(gameState, action)`: Validate player action
- `processMove(gameState, move)`: Apply move to game state
- `checkWinCondition(player)`: Determine if player has won
- `getNextPlayer(current)`: Calculate next player's turn
