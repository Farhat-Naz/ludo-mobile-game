# Dice Module

Dice rolling logic and validation for the Ludo game.

## Purpose

- Generate random dice rolls (1-6)
- Validate dice values
- Determine extra turn conditions (rolling 6)
- Track consecutive rolls

## Key Functions

- `rollDice()`: Generate random dice value
- `isExtraTurn(value)`: Check if roll grants extra turn
- `canOpenToken(value)`: Check if roll allows token to exit home
