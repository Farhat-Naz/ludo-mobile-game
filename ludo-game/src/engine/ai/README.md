# AI Module

AI opponent logic and strategy implementation.

## Purpose

- Implement AI decision-making
- Evaluate move options
- Support difficulty levels (easy, medium, hard)
- Simulate player strategies

## Difficulty Levels

- **Easy**: Random valid moves
- **Medium**: Basic strategy (prioritize capturing, finishing)
- **Hard**: Advanced planning and position evaluation

## Key Functions

- `selectMove(gameState, difficulty)`: Choose best move for AI
- `evaluatePosition(gameState)`: Score current game state
- `predictOpponentMove(gameState)`: Anticipate opponent actions
