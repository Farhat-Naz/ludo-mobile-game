# Animations

Animation utilities and configurations using React Native Reanimated v3.

## Purpose

- Provide smooth 60 FPS animations
- Centralize animation configurations
- Ensure consistent animation timings

## Animation Types

### Token Animations
- Token movement along path
- Token capture effects
- Token entrance/exit

### Dice Animations
- Dice roll animation
- Value reveal transition

### Screen Transitions
- Fade in/out
- Slide transitions
- Modal animations

## Key Utilities

- `tokenMoveAnimation(from, to)`: Animate token movement
- `diceRollAnimation()`: Animate dice roll
- `captureEffect(token)`: Animate token capture
