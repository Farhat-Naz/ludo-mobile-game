# UI Components

Reusable React Native components for the Ludo game.

## Purpose

- Provide reusable UI building blocks
- Ensure consistent design across the app
- Implement colorblind-accessible visuals

## Component Categories

### Board Components
- `BoardView`: Main game board layout
- `PathCell`: Individual board cells
- `SafeZone`: Safe zone areas

### Token Components
- `Token`: Animated token piece
- `TokenCounter`: Token count display

### Dice Components
- `DiceView`: Animated dice
- `DiceButton`: Rollable dice button

### Common Components
- `Button`: Primary button
- `Card`: Card container
- `Modal`: Modal dialog

### Player Components
- `PlayerAvatar`: Player icon
- `ScoreBoard`: Current scores
